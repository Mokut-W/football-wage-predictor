# Football Player Wage Predictor

A machine learning app that predicts a football player's weekly wage based on their FIFA 22 stats.

## Live Demo
[Try it on Hugging Face Spaces](https://huggingface.co/spaces/mokut/football-wage-predictor)

## Model
- Algorithm: Gradient Boosting
- Dataset: FIFA 22 (17,041 players)
- MAE: €4,495 per week

## Features Used
- Age, Overall Rating, Potential, Market Value
- Pace, Shooting, Passing, Dribbling, Defending, Physic
- Engineered: Overall per Age, Potential Gap, Attacking Average, Defensive Average, Overall Squared
